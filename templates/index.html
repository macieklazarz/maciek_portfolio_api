About me.


Hi! My name is Maciek

Currently I work as a backend developer in IQVIA Technology Solutions. I am part of the team responible for supporing CRM application developed by my company.


ETL

My daily task are mainly related to development of ETL tools on behalf of dedicated software, compatibile with mentioned CRM. I create tools responible for bilateral dataflow between application databases and ftp service. Another tool supporting my work is Informatica PwC.


Oracle

As a backend developer focused on CRM application I spend a lot of time searching the info among large data content collected by various clients. Malfuncioned ETL tools reqiures investigations to let me seek the source of problem, solve it and make the tool efficient. Another aspect is boosting existing tools. Python

Some solutions applied by my are supported by tools developed in Python. This programming language helps me in situations when ETL configurations can be replaced by customized Python script. As there are libraries dedicated to manipulate the data some tasks are possible to be completed quicker.
